# Firebase Studio - InsightFlow

This is a NextJS starter in Firebase Studio, optimized for neuro-divergent trading focus.

## 🛠️ Architecture
- **Dashboard**: Multiple view modes (Focus, Dual, Quad, Pro).
- **Neuro Engine**: 16 medical-consultant-approved profiles.
- **Physics**: Dynamic chart rendering via `chartPhysics.ts`.

Take a look at `src/app/page.tsx` to launch the Social Platform, or navigate to `/dashboard` for the trading view.
